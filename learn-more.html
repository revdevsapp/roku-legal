Retro Media Client is a personal Roku channel designed to browse and play media from a user-controlled media server on your local network.

What This App Does

  • Browse movies and TV shows from your own media library
  • Navigate TV shows by season and episode
  • Shuffle movies, shows, seasons, or your entire library
  • Play media directly from your server to your Roku device

What This App Does Not Do

  • It does not provide or host media content
  • It does not stream from third-party services
  • It does not include ads or subscriptions
  • It does not collect personal data or analytics

How It Works

The app connects to a media server that you configure separately. The server supplies media metadata (titles, posters, episodes) and stream URLs. The Roku app acts purely as a playback and navigation interface.

Network Requirements

  • Your Roku device and media server must be on the same local network
  • The server must be running and reachable by IP address or hostname
  • The server must expose compatible streaming URLs

Supported Media

For best compatibility, media should be encoded in formats supported by Roku devices (such as H.264 video with AAC audio in MP4 containers).

Privacy First

Retro Media Client does not collect user information, track usage, or transmit data outside your local environment. All media access remains under your control.

Retro Media Client is intended for personal use with user-owned or user-managed media libraries.
