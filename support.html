Support

Retro Media Client is a personal Roku channel that connects to a user-managed media server on your local network.

Quick Start
  • Confirm your Roku is on the same network as your media server.
  • Verify the server base URL configured in the app is correct (example: http://192.168.1.20:32400).
  • Make sure your server is running and reachable from another device on your network.

Troubleshooting

Nothing loads / empty library
  • Check the server URL and port.
  • Confirm your server exposes the required API endpoints.
  • If your server requires authentication, ensure your server is configured to allow access from the Roku app.
Playback fails
  • Confirm the media server provides a direct stream URL reachable by the Roku.
  • Test playback of the same URL from another device on your network.
  • Verify your media format is Roku-compatible (H.264/AAC in MP4 is most compatible).
Posters/metadata missing
  • Confirm the API response includes poster URLs (recommended: HTTPS or LAN-accessible HTTP).
  • Check that poster image URLs are reachable from the Roku device.

Contact

For support requests, include:

  • Roku model
  • Server type (custom / Plex / Jellyfin)
  • Server URL format (do not include passwords)
  • What screen you were on + what happened
